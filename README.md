# Personal Assistant

Personal assistant system for family management with multiple specialized skills.

## Skills

### 1. Basket Manager
- Grocery shopping management
- Price comparison between stores
- Shopping lists
- Colombian supermarkets (Mercar, Ara, D1, Cañaveral, etc.)

### 2. Meal Planner
- Weekly menu planning
- Recipe management
- Nutrition tracking
- Shopping list from menu

### 3. Finance Tracker
- Income/expense tracking
- Budget management
- Savings goals

### 4. Personal Routine
- Habit tracking
- Daily routines
- Streaks system

## Quick Start

```bash
# Install dependencies
npm install

# Initialize database
npm run db:init

# Start server
npm run dev
```

## API

Server runs on `http://localhost:3003`
