# SRC.md - personal-assistant

> Asistente personal con base de datos - SouthLabs

## Proyecto
- **Nombre:** personal-assistant
- **Tipo:** API REST (Node.js + Express)
- **Stack:** Express, TypeScript, SQLite (better-sqlite3)

## Funcionalidad
Sistema de gestión de asistente personal con almacenamiento local.

## Estructura
```
personal-assistant/
├── db/          # Base de datos SQLite
├── services/    # Lógica de negocio
```

## Scripts
| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Desarrollo (tsx watch) |
| `npm run build` | Compilar TypeScript |
| `npm run db:init` | Inicializar DB |
